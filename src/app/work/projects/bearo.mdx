## Overview

Bearo brings the simplicity of Venmo to crypto payments. Send stablecoins to anyone using just their username ‚Äî no wallet addresses to copy, no chain confusion, just simple peer-to-peer payments.

**Status: Currently in Development** ‚Äî Follow the progress on our [24/7 livestream](https://pump.fun/coin/FdFUGJSzJXDCZemQbkBwYs3tZEvixyEc8cZfRqJrpump).

## Key Features

- **Username-Based Payments**: Send USDC/USDT to users by @username, not wallet addresses
- **Multi-Chain Support**: Ethereum, Polygon, and Base with Base as default for lowest fees
- **Email Authentication**: thirdweb user wallets with email verification ‚Äî no seed phrases needed
- **Smart Balance Filtering**: View balances by chain and token with persistent user preferences
- **Real-Time Tracking**: Transaction status monitoring with blockchain confirmation updates
- **Mobile-First Design**: Venmo-inspired UI optimized for mobile devices

## Technical Stack

- **Frontend**: React 18 + TypeScript + Vite
- **Styling**: Tailwind CSS with custom Venmo-inspired design system
- **Database**: Supabase (PostgreSQL) with Row Level Security
- **Blockchain**: thirdweb API v1 for all Web3 operations
- **Authentication**: thirdweb email verification + JWT token management

## Payment Flow

1. **User Search**: Look up recipient by username in Supabase
2. **Payment Creation**: Call thirdweb `/v1/payments` endpoint
3. **Payment Execution**: Complete via `/v1/payments/{id}` endpoint
4. **Transaction Recording**: Store details in Supabase
5. **Real-time Updates**: Status updates via Supabase subscriptions
